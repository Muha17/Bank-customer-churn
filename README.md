# Bank-customer-churn
Analysis of retail banking customer data using SQL to identify churn drivers, quantify retention risk and deliver actionable insights
# Overview
This project analyzes customer churn in a retail banking environment using SQL for data preparation and analysis, and Power BI for visualization. The objective is to identify churn drivers, assess customer risk, and deliver actionable insights that support retention strategies.
# Business Objective
Customer attrition directly impacts revenue and long-term profitability in banking. This analysis helps stakeholders understand:
Who is churning
Which customer segments are most at risk
Where revenue exposure is highest
What actions can improve retention

# Key Insights

Inactive customers churn at significantly higher rates

Early-tenure customers represent the highest churn risk

High-balance customer churn poses material revenue exposure

Multi-product customers are more likely to be retained

Churn behavior varies by geography

# Business Recommendations

Target inactive and early-tenure customers with retention campaigns

Proactively engage high-value customers

Encourage product bundling to improve retention

Investigate regional churn drivers and tailor interventions

# Conclusion

This project demonstrates how SQL-driven customer analytics combined with Power BI visualization can uncover churn drivers and inform effective retention strategies in a banking environment.
